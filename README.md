# implementation-du-modele-gaussien-coalescent
travail réalisé dans le cadre du stage de master 1 bioinformatique et biostatistiques

Il s'agit d'ajouter le modèle gaussien, en plus des modèles OU, EB et BM, à l'analyse réalisée dans l'article "Evaluating the Performance of Widely Used Phylogenetic Models for Gene Expression Evolution" nov, 2023. 

Le dossier data and trees contient les jeux de données et les arbres des espèces associés.
Le dossier script contient les scripts R. Pour le moment, le dossier moyennes contient les scripts d'analyse prenant uniquement en compte les moyennes des traits par espèces. Le dossier moy+ind contient un début d'analyse pour prendre en compre la variance entre individus.
